# Wrestling Club CRM

A complete, free, and easy-to-use website and management system for wrestling clubs.

![Wrestling Club CRM](https://img.shields.io/badge/Version-1.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Platform](https://img.shields.io/badge/Platform-Web-orange)

---

## 🎯 Features

### Public Website
- ✅ **Homepage** - Slideshow, welcome message, club information
- ✅ **About Page** - Coach bios, club history, achievements
- ✅ **Schedule Page** - Practice times, tournament calendar
- ✅ **Contact Form** - Let parents reach out easily
- ✅ **Registration Form** - Online signups with digital waiver
- ✅ **Mobile Responsive** - Works perfectly on phones and tablets

### Admin Dashboard
- ✅ **Registration Management** - View, search, export all signups
- ✅ **Team Roster** - Manage active wrestlers, add/remove members
- ✅ **Contact Messages** - View and respond to inquiries
- ✅ **Content Management** - Update schedule, slideshow, audio
- ✅ **Bulk Email** - Send updates to selected team members
- ✅ **Data Export** - Download registrations as CSV

### Automation
- ✅ **Email Notifications** - Instant alerts for new registrations and contacts
- ✅ **Auto-Save** - All data automatically saved to GitHub
- ✅ **Version Control** - Full history of all changes
- ✅ **Zero Maintenance** - No database to manage

## 📊 What Data is Collected?

### Registration Form
- Wrestler name, DOB, grade, weight
- Parent/guardian name, email, phone
- Address and town
- Waiver acknowledgment timestamp

### Contact Form
- Name, email, message
- Submission timestamp

### Team Roster
- Wrestler name, grade, weight
- Parent email and phone
- Notes (optional)

**All data is stored in your GitHub repository - you have full control.**
