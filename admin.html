First-time setup (read-only)

  1. Create a free project at supabase.com
  2. In Settings → API, copy your Project URL and anon public key
  3. In the SQL Editor, run the SQL included at the bottom of this file (expand source) to create tables and policies
  4. In Auth → Users, create a user account with your email
  5. Insert your email into the admins table
  6. Set the two constants in this page to your Supabase values and deploy
